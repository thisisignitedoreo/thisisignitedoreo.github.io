How to partition drive for use with Windows WIM Installer

  1. Clean and convert drive
  2. Create BOOT Partition
  3. Create Primary (system) Partition

1. Clean drive

Warning! This will delete everything from drive!

1.1. Get list of all of your disks
DISKPART> list disk
 Example output:
Disk ### Status Size
-------- ------ ----
Disk 0 Online 40gb
 1.2. Select this disk
DISKPART> sel disk 0
 1.3. Clean it
DISKPART> clean
 1.3. Convert to GPT
DISKPART> convert gpt

2. Create BOOT Partition

2.1. Create partition
DISKPART> create part efi size=500
 2.2. Format it to FAT32
DISKPART> format fs=fat32 quick
 2.3. Assign letter to it (remember it, installer will ask for it)
DISKPART> assign letter w

3. Create Primary (system) Partition

3.1. Create partition
DISKPART> create part primary
 3.2. Format it to NTFS
DISKPART> format fs=ntfs quick
 3.3. Assign letter to it (remember it, installer will ask for it)
DISKPART> assign letter c
